臺北 e 大學習輔助程式 v0.1.2（Windows 64-bit）

使用方式
1. 完整解壓縮 ZIP，不能只取出 EXE。
2. 確認電腦已安裝正式版 Google Chrome。
3. 執行 TaipeiELearnAssistant.exe。
4. 第一次使用，在程式開啟的 Chrome 手動登入臺北 e 大。

環境需求
- Windows 10/11 64-bit
- Google Chrome
- 不需安裝 Python
- 操作臺北 e 大仍需網路

登入 profile 與日誌位於：
%LOCALAPPDATA%\TaipeiELearnAssistant

請勿分享 chrome-profile 資料夾，內含登入工作階段。
