臺北 e 大學習輔助程式 v1.1.0（Windows 64-bit）

安裝
1. 完整解壓縮 ZIP，不能只取出 EXE。
2. 確認電腦已安裝正式版 Google Chrome。
3. 執行 TaipeiELearnAssistant.exe。
4. 第一次使用，在程式開啟的 Chrome 手動登入臺北 e 大。

主要功能
- 學習紀錄掃描、課程倒數及連續上課
- 額外時數補正
- 單頁單選／多選測驗輔助與自動送出
- 每列可設定 1～5 門、可保存與收合面板的批次選課

環境需求
- Windows 10/11 64-bit
- Google Chrome
- 操作臺北 e 大時需要網路
- 不需安裝 Python

登入 profile、設定與日誌位於：
%LOCALAPPDATA%\TaipeiELearnAssistant

請勿分享 chrome-profile 資料夾；其中可能包含有效登入工作階段。
