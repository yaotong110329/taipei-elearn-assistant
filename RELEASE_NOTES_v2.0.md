# v2.0 - 自動答題與批次選課

## 新增

- 批次選課：常用關鍵字、跨頁搜尋、每個關鍵字最多 5 門、排除已報名及認證時數 0 小時課程。
- 測驗佇列：掃描可答課程、勾選、擷取單頁單選／多選題、AI 答案驗證、自動填入及送出。
- 上課額外時數補正。
- 任何階段返回學習紀錄，以及 Chrome 被手動關閉後重新開啟。

## 改善

- 認證時數按 50% 計算實際上課時間，倒數結束直接銜接下一門。
- 改善一般課程、兩段式課程及 SCORM 多層入口的切換。
- 課程離開後自動處理可能出現的問卷視窗。
- GUI 顯示目前課程門數、總數及倒數時間。

## 安裝

下載 `TaipeiELearnAssistant-v2.0-win64.zip`，完整解壓縮後執行 `TaipeiELearnAssistant\TaipeiELearnAssistant.exe`。需要 Windows 10/11 64-bit 及正式版 Google Chrome，不需安裝 Python。
